# Manual do Usuário - Sistema do Dizimo do Fatima  v1.3.6

Bem-vindo ao manual oficial do seu novo sistema de gestão. Este documento foi criado para ajudar você e sua equipe a operar o sistema com eficiência e segurança.

---

## 1. Acesso ao Sistema
Para entrar no sistema, acesse o endereço fornecido e insira seu **Usuário** e **Senha**. 
- Se você for um **Administrador (Staff)**, terá acesso à gestão de usuários e ao painel completo do Django.
- **Dica de Segurança**: Nunca compartilhe sua senha pessoal. Cada agente deve ter seu próprio login para que as auditorias sejam precisas.

## 2. Gestão de Dizimistas
O coração do sistema é o cadastro de fiéis.
- **Adicionar Novo**: Clique em "Adicionar Dizimista". 
- **Número de Identificação**: O sistema permite que você atribua um número único para cada dizimista.
- **Cônjuge Dinâmico**: Ao selecionar o estado civil "Casado(a)", um novo campo aparecerá automaticamente para você registrar o nome do cônjuge.
- **Busca e Filtros**: Use a barra de busca no topo da lista para encontrar alguém pelo nome ou número rapidamente.

## 3. Registro de Pagamentos
Registrar uma contribuição é simples e rápido:
1. Vá na lista de Dizimistas ou na aba Pagamentos.
2. Clique em "Registrar Pagamento".
3. **Data Automática**: O sistema já preenche a data de hoje para você, mas você pode conferir o valor.
4. **Histórico Imutável**: Para garantir a segurança financeira, o sistema **não permite a exclusão** de pagamentos pela interface comum, evitando erros ou fraudes.

## 4. O Dashboard e Filtros de Meses
O Dashboard oferece uma visão clara da saúde financeira:
- **Cards de Status**: Mostram o arrecadado Hoje, na Semana e no Mês.
- **Filtro de Histórico**: No topo do dashboard, você pode selecionar **meses anteriores**. Ao escolher um mês passado, os cartões de "Hoje" somem para você focar apenas no total daquele mês específico.
- **Gráficos**: As barras mostram a evolução das contribuições visualmente.

## 5. Exportação para Excel
Sempre que precisar de um relatório físico ou para prestação de contas:
1. Vá ao Dashboard ou à Lista de Pagamentos.
2. Clique no botão **Exportar**.
3. O sistema gerará um arquivo `.xlsx` (Excel) com todos os dados organizados, prontos para impressão.

## 6. Importação em Massa
Se você tiver uma lista grande de dizimistas em uma planilha Excel, você pode usar a função de **Importar Excel** na lista de dizimistas para cadastrar todos de uma vez, economizando horas de trabalho manual.
